　　【eNet硅谷动力消息】全球最大的软件厂商微软公司宣布，今年夏季将在南非、博茨瓦纳、 纳米比亚和尼日利亚等非洲国家推出 Windows XP Starter Edition 操作系统。
　　上周微软公司表示，它的功能受到限制、价格低廉的入门级 Windows操作系统 已经在拉丁美洲、欧洲、中东和亚洲的八十多个国家和地区推出，微软公司计划今年六月在非洲一些国家发布这一入门级操作系统。
　　在开始向非洲提供 祖鲁语（isiZulu） Windows XP软件包之后一周微软公司宣布了上述的决定，祖鲁语是南非最大的语言，24%南非人的母Z是祖鲁语。微软公司计划提供三种南非语言的软件包，祖鲁语软件包是最先提供的非 洲语种。
　　南非开源软件Ubuntu Linux distribution的倡导者和创始人Mark Shuttleworth向媒体表示，微软公司正在和开源软件在非洲的成功进行竞争。他说：“我们赞同微软公司的举措，它已经认可采用当地语言普及 IT 技术的重大意义。我们希望它们进一步努力，追赶开源社区已经完成的工作。”
　　Shuttleworth表示，微软公司仅仅发布了一个南非语言的软件包，非赢利组织 Translate.org.za 已经转化了开源办公应用软件，OpenOffice.org 已经采用了十一种南非的官方语言。这个组织同时还在进行开发采用不同南非语言 Linux 台式机环境的 KDE 和 GNOME以及 Firefox浏览器。
　　Shuttleworth指出，尽管微软公司的 Windows XP 入门级操作系统的功能类似于XP的完整版本，但它受到了多种限制，例如 XP 入门级操作系统只有三个应用软件，每个应用软件只能够同时运行三个窗口，
　　仅仅配置了低端处理器，例如英特尔公司的赛扬芯片或AMD公司的Duron芯片。
　　Shuttleworth 认为微软公司应该取消这些限制，他说：“我们希望微软公司取消内置在入门级操作系统中的这些限制，这样能够获得目前硬件的许多优势，在一定时间内使用户获得更多的应用。”
